# IPL Data Analysis (EDA Project)

## 📌 Overview
This project focuses on performing Exploratory Data Analysis (EDA) on IPL (Indian Premier League) dataset to uncover patterns, trends, and insights related to team performance, match outcomes, and player statistics.

## 🎯 Objective
The main objective of this project is to analyze IPL data and extract meaningful insights that can help in understanding match-winning factors and decision-making strategies.

## 📊 Dataset
The dataset contains information about IPL matches including:
- Teams
- Match results
- Venue details
- Runs, wickets, toss decisions

## 🛠 Tools & Technologies
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

## 🔍 Key Insights
- Team batting second(chasing)win majority of matches(~55-60%)
- Toss decision plays a significant role in match outcome.
- Certain venues show clear patterns favouring batting or bowling team.
- Team proformence varies significantly across seasons and conditions.

## 📈 Analysis Performed
- Data Cleaning and Preprocessing
- Handling missing values
- Data Visualization
- Trend Analysis

## 🧠 Conclusion
This analysis highlights key factors that influence IPL match outcomes. Teams can use these insights to make better strategic decisions such as choosing to chase or selecting playing venues wisely.
